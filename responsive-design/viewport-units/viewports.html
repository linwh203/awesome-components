How to set 100 view height ?

100vh has issue on mobile, mobile doesn't count address bar into 100vh

solution No.1: using dvh unit. set to 100dvh, but the page will re-adjust dynamic height to re-render for 1 second.

solution No.2: using svh unit. set to 100svh. s means always using small view port(assuming address bar or other stuff always be there) size

svh not supported on desktop very well, a good way is set height:

height: 100vh;
height: 100svh;

there also have lvh, means large view port size

for font-size, a better way is using clamp unit
